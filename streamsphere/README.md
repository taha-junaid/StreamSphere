## Streamsphere backend

- This is a Spring boot app (Java) that runs on EC2  
- It manages channel creation, connections, and facilitates the host and participants to communicate to synchronize their video playback times as `SyncEvent`, get live participants in a channel, view recently watched movies on home page etc.
